### 문제 설명

<p>같은 차종의 빨간색 자동차 $A$대, 초록색 자동차 $B$대, 파란색 자동차 $C$대를 $N$칸의 주차장에 $1$대씩 주차하려고 한다.</p>

<p>모든 자동차를 한 칸에 한 대씩 주차할 수 있는 경우의 수를 구하라.</p>
